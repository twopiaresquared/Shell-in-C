# Shell-in-C
Name: Shreyas Rane
RUID: 189006029

A simple shell in C
1)Executes basic commands like cd, pwd, clear, exit.
2)The programs are invoked using forking and child processes.
3)Use of dup2 for the following:
  <cmd> <args> <output>
  <cmd> <args> <input> <output>
4) pipe and dup2 for multiple piping
